
# MyDietApp

### 기능

1. 달력
	* 날짜별 몸무게,식사량,운동량 데이터 저장/수정/삭제
2. 차트
	* 일주일,한달,3개월,1년별 데이터 모아 보기
	* 차트를 bmi 구간별로 나누기
3. 설정
	*	알림 설정
	*	모든 데이터 삭제
	*	앱 정보
